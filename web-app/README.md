# Bin Day Brain - Web App

Wollongong Smart Bin Reminders - Web version

## Features

- Find your address and see bin collection dates
- Color-coded cards (FOGO, Recycling, Landfill)
- Weather alerts for windy/rainy days
- "Which Bin?" A-Z waste guide search
- Works offline (PWA)
- Installable on mobile devices

## Hosting on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to Settings > Pages
3. Select "main" branch and save
4. Access at `https://yourusername.github.io/repository-name/`

## Local Development

Just open `index.html` in a browser, or use a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Then open http://localhost:8000
